# ATS Resume (LaTeX)

This repository contains my ATS-friendly resume built using LaTeX and Overleaf.

## Structure

```
main.tex
sections/
assets/
output/
```

## Features

- One-page ATS-friendly resume
- Modular LaTeX structure
- Easy to maintain and update
- Version controlled with Git

## Output

The latest compiled resume is available in:

```
output/Dhimahi_Mehta_Resume.pdf
```
